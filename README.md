# VibraSonic 🎵

Reproductor de música web con DJ automático, letras sincronizadas, visualizador y más.

## 🚀 Opción 1: Abrir directo (rápido)

Descarga `vibrasonic.html` y ábrelo con doble clic en tu navegador.

## 🚀 Opción 2: Desplegar en Vercel (recomendado)

### Paso 1: Crear repositorio en GitHub
1. Ve a [github.com](https://github.com) y crea un repositorio nuevo (ej: `vibrasonic`)
2. Sube estos 3 archivos manteniendo la estructura:
   ```
   index.html
   css/styles.css
   js/app.js
   ```

### Paso 2: Conectar con Vercel
1. Ve a [vercel.com](https://vercel.com) e inicia sesión con GitHub
2. Click en **"Add New Project"**
3. Selecciona tu repositorio `vibrasonic`
4. Click en **"Deploy"**
5. ¡Listo! Vercel te dará una URL como `vibrasonic.vercel.app`

### Paso 3: Actualizar (cuando haya cambios)
1. Descarga los 3 archivos actualizados
2. Súbelos a GitHub (reemplazando los anteriores)
3. Vercel se actualiza automáticamente en segundos

## ✨ Funciones

- ✅ Subir música (MP3, FLAC, WAV, M4A, etc.)
- ✅ Letras sincronizadas automáticas (LRCLIB)
- ✅ Modo DJ con crossfade y análisis de BPM
- ✅ Modo oscuro / claro
- ✅ Visualizador (barras, onda, circular)
- ✅ Playlists y favoritos
- ✅ Compartir canciones
- ✅ Responsive (móvil, tablet, PC)
- ✅ Las canciones se guardan (IndexedDB)
